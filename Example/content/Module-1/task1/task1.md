# Задача 1.1. Сумма двух чисел

* **Тема:** Ввод-вывод, арифметика
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны два целых числа $a$ и $b$. Выведите их сумму.

## Формат входных данных
В единственной строке записаны два целых числа $a$ и $b$ ($-10^9 \le a, b \le 10^9$), разделённые пробелом.

## Формат выходных данных
Выведите одно целое число — сумму $a + b$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 3 5 | 8 |
