# Задача 1.3. Площадь прямоугольника

* **Тема:** Арифметика
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны длины сторон прямоугольника $a$ и $b$. Вычислите его площадь.

## Формат входных данных
В единственной строке записаны два целых числа $a$ и $b$ ($1 \le a, b \le 10^4$) — длины сторон прямоугольника.

## Формат выходных данных
Выведите одно целое число — площадь прямоугольника.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 3 7 | 21 |
