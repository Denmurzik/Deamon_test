# Ввод-вывод и базовые арифметические операции

## 1. Стандартный ввод и вывод

Любая программа взаимодействует с внешним миром через **ввод** (input) и **вывод** (output). В задачах олимпиадного программирования чаще всего используется чтение из файла `input.txt` и запись в `output.txt`.

### Пример на C:
```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}
```

### Пример на Python:
```python
a, b = map(int, input().split())
print(a + b)
```

## 2. Базовые арифметические операции

| Операция | C/C++ | Python |
|----------|-------|--------|
| Сложение | `a + b` | `a + b` |
| Вычитание | `a - b` | `a - b` |
| Умножение | `a * b` | `a * b` |
| Целочисленное деление | `a / b` | `a // b` |
| Остаток от деления | `a % b` | `a % b` |

## 3. Типы данных

- `int` — целое число (до $\pm 2 \cdot 10^9$)
- `long long` (C/C++) — длинное целое (до $\pm 9 \cdot 10^{18}$)
- `double` — число с плавающей точкой
- В Python ограничения на размер целых чисел нет

## 4. Математические функции

Для вычислений с вещественными числами используйте библиотеку `math.h` (C) или модуль `math` (Python):

```c
#include <math.h>
double d = sqrt(x*x + y*y);  // расстояние
```

```python
import math
d = math.sqrt(x**2 + y**2)
```
