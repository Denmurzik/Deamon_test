# Задача 1.9. Расстояние между точками

* **Тема:** Арифметика, формулы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны координаты двух точек на плоскости $(x_1, y_1)$ и $(x_2, y_2)$. Вычислите евклидово расстояние между ними.

$$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

## Формат входных данных
В единственной строке записаны четыре целых числа $x_1$, $y_1$, $x_2$, $y_2$ ($-10^4 \le x_i, y_i \le 10^4$).

## Формат выходных данных
Выведите вещественное число — расстояние между точками с точностью 6 знаков после запятой.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 0 0 3 4 | 5.000000 |
