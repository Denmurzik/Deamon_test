# Задача 1.6. Перевод секунд

* **Тема:** Арифметика, деление с остатком
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано количество секунд $S$. Переведите его в часы, минуты и секунды.

## Формат входных данных
В единственной строке записано целое число $S$ ($0 \le S \le 86400$).

## Формат выходных данных
Выведите три числа через пробел: часы, минуты и секунды.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 3661 | 1 1 1 |
