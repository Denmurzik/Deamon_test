# Задача 2.5. Факториал числа

* **Тема:** Циклы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано неотрицательное целое число $N$. Вычислите $N!$ (факториал числа $N$).

$$N! = 1 \cdot 2 \cdot 3 \cdot \ldots \cdot N$$

По определению, $0! = 1$.

## Формат входных данных
В единственной строке записано число $N$ ($0 \le N \le 20$).

## Формат выходных данных
Выведите значение $N!$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 5 | 120 |
