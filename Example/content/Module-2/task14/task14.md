# Задача 2.4. Сумма от 1 до N

* **Тема:** Циклы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано натуральное число $N$. Вычислите сумму всех натуральных чисел от 1 до $N$ включительно.

## Формат входных данных
В единственной строке записано натуральное число $N$ ($1 \le N \le 10^6$).

## Формат выходных данных
Выведите одно число — сумму от 1 до $N$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 5 | 15 |

## Примечания
Можно использовать формулу $\frac{N(N+1)}{2}$ или цикл.
