# Задача 2.7. Проверка простоты числа

* **Тема:** Циклы, условия
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано натуральное число $N$. Определите, является ли оно простым.

Число называется простым, если оно больше 1 и делится только на 1 и на себя.

## Формат входных данных
В единственной строке записано натуральное число $N$ ($1 \le N \le 10^9$).

## Формат выходных данных
Выведите `YES`, если число простое, и `NO` в противном случае.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 7 | YES |
| 4 | NO |

## Примечания
Для эффективной проверки достаточно перебирать делители до $\sqrt{N}$.
