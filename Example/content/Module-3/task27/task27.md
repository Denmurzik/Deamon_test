# Задача 3.7. Длина строки

* **Тема:** Строки
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дана строка $S$, состоящая из строчных латинских букв. Выведите её длину.

## Формат входных данных
В единственной строке записана строка $S$ ($1 \le |S| \le 1000$).

## Формат выходных данных
Выведите одно число — длину строки.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| hello | 5 |
