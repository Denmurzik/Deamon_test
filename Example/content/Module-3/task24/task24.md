# Задача 3.4. Подсчёт вхождений

* **Тема:** Массивы, подсчёт
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел и число $X$. Подсчитайте, сколько раз $X$ встречается в массиве.

## Формат входных данных
В первой строке записаны два числа $N$ и $X$ ($1 \le N \le 10^5$, $-10^9 \le X \le 10^9$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$).

## Формат выходных данных
Выведите одно число — количество вхождений $X$ в массив.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **7 3**<br>1 3 3 2 3 5 3 | **4** |
