# Задача 3.3. Линейный поиск

* **Тема:** Массивы, поиск
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел и число $X$. Найдите позицию первого вхождения $X$ в массив (нумерация с 1). Если элемент не найден, выведите $-1$.

## Формат входных данных
В первой строке записаны два числа $N$ и $X$ ($1 \le N \le 10^5$, $-10^9 \le X \le 10^9$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$).

## Формат выходных данных
Выведите позицию первого вхождения $X$ (начиная с 1) или $-1$, если элемент не найден.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **5 3**<br>1 2 3 4 5 | **3** |
