# Задача 3.9. Подсчёт слов

* **Тема:** Строки
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дана строка, состоящая из слов, разделённых ровно одним пробелом. Подсчитайте количество слов в строке.

## Формат входных данных
В единственной строке записана непустая строка длины не более 1000 символов.

## Формат выходных данных
Выведите количество слов.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| hello world foo | 3 |
