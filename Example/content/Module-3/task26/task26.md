# Задача 3.6. Сортировка пузырьком

* **Тема:** Массивы, сортировка
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 2.0 секунды
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел. Отсортируйте его по неубыванию методом пузырьковой сортировки. Использование встроенных функций сортировки запрещено.

## Формат входных данных
В первой строке записано число $N$ ($1 \le N \le 10^4$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$).

## Формат выходных данных
Выведите отсортированный массив через пробел.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **5**<br>5 3 1 4 2 | **1 2 3 4 5** |

## Примечания
Алгоритм пузырьковой сортировки многократно проходит по массиву, сравнивая соседние элементы и меняя их местами, если они стоят в неправильном порядке. Сложность: $O(N^2)$.
