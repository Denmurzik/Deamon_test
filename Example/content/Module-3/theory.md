# Массивы и строки

## 1. Массивы

Массив — это упорядоченная коллекция элементов одного типа с доступом по индексу за $O(1)$.

### Объявление и использование (C):
```c
int arr[100];
int n;
scanf("%d", &n);
for (int i = 0; i < n; i++) {
    scanf("%d", &arr[i]);
}
```

### В Python:
```python
n = int(input())
arr = list(map(int, input().split()))
```

## 2. Основные операции с массивами

- **Обход**: перебор всех элементов за $O(N)$
- **Поиск**: линейный поиск за $O(N)$
- **Сортировка**: пузырьковая $O(N^2)$, быстрая $O(N \log N)$
- **Разворот**: обмен элементов с краёв к центру за $O(N)$

```c
// Разворот массива
for (int i = 0; i < n / 2; i++) {
    int tmp = arr[i];
    arr[i] = arr[n - 1 - i];
    arr[n - 1 - i] = tmp;
}
```

## 3. Строки

В C строка — массив символов `char[]`, заканчивающийся нулевым символом `'\0'`.

```c
char s[101];
scanf("%s", s);
int len = strlen(s);  // длина строки
```

В Python строки — неизменяемые объекты:
```python
s = input()
print(len(s))
```

## 4. Основные операции со строками

| Операция | C (`string.h`) | Python |
|----------|----------------|--------|
| Длина | `strlen(s)` | `len(s)` |
| Сравнение | `strcmp(s1, s2)` | `s1 == s2` |
| Копирование | `strcpy(dst, src)` | `dst = src` |
| Конкатенация | `strcat(s1, s2)` | `s1 + s2` |

## 5. Посимвольная обработка

```c
for (int i = 0; s[i] != '\0'; i++) {
    if (s[i] >= 'a' && s[i] <= 'z') {
        s[i] = s[i] - 'a' + 'A';  // в верхний регистр
    }
}
```
