# Задача 3. Быстрая сортировка

* **Тема:** Разделяй и властвуй, алгоритмы сортировки
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 64 МБ

## Условие задачи
Вам необходимо написать программу, которая сортирует заданный массив целых чисел по неубыванию, используя алгоритм **Быстрой сортировки (Quick Sort)** Хоара.

В данной задаче вам необходимо самостоятельно реализовать выбор опорного элемента (pivot) и разбиение массива (partition). Подумайте о том, как избежать худшего случая $O(N^2)$ при выборе опорного элемента (например, используйте случайный выбор опорного элемента или медиану из трех).

## Формат входных данных
В первой строке входного файла задано одно целое число $N$ ($1 \le N \le 10^5$) — размер массива.
Во второй строке записаны $N$ целых чисел $a_i$, по модулю не превосходящих $10^9$ — элементы массива.

## Формат выходных данных
В выходной файл выведите $N$ чисел, разделенных пробелом — элементы исходного массива, отсортированные по неубыванию.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **6**<br>10 -5 3 8 0 3 | **-5 0 3 3 8 10** |

## Примечания
Обратите внимание, что массив может содержать дубликаты. Ваша реализация функции `partition` должна корректно обрабатывать элементы, равные опорному, чтобы алгоритм не работал слишком долго на массивах, состоящих из одинаковых чисел.
