# Задача 2. Сортировка слиянием

* **Тема:** Разделяй и властвуй, алгоритмы сортировки
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.5 секунды
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Требуется отсортировать заданный массив целых чисел по неубыванию.
Для решения этой задачи необходимо реализовать алгоритм **Сортировки слиянием (Merge Sort)**. Использование встроенных функций сортировки (таких как `qsort` в C или `std::sort` в C++) в данной задаче строго запрещено. Ваша программа должна работать за время $O(N \log N)$ в худшем случае.

## Формат входных данных
В первой строке входного файла задано целое число $N$ ($1 \le N \le 10^5$) — количество элементов в массиве.
Во второй строке заданы $N$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$) — элементы массива, разделенные пробелом.

## Формат выходных данных
Выведите в выходной файл элементы отсортированного массива через пробел.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **5**<br>5 3 2 4 1 | **1 2 3 4 5** |

## Примечания
Алгоритм сортировки слиянием базируется на парадигме «разделяй и властвуй». Он рекурсивно делит массив пополам, пока размер подмассива не станет равным 1, а затем сливает отсортированные подмассивы вместе. Обратите внимание на выделение дополнительной памяти для слияния.
