# Задача 3B. Кратчайший путь (BFS)

Дан невзвешенный неориентированный граф. Найдите кратчайшее расстояние от вершины 1 до вершины N.

## Входные данные
- Первая строка: два числа N и M
- Следующие M строк: два числа u, v — ребро

## Выходные данные
Одно число — кратчайшее расстояние. Если путь не существует, выведите -1.

## Пример
**Вход:**
```
4 4
1 2
2 3
3 4
1 3
```
**Выход:**
```
2
```
