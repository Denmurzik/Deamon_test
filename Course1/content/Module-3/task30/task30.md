# Задача 3.10. Подстрока

* **Тема:** Строки
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны две строки $S$ и $T$, состоящие из строчных латинских букв. Определите, входит ли $T$ в $S$ как подстрока.

## Формат входных данных
В первой строке записана строка $S$ ($1 \le |S| \le 10^5$).
Во второй строке записана строка $T$ ($1 \le |T| \le |S|$).

## Формат выходных данных
Выведите `YES`, если $T$ является подстрокой $S$, и `NO` в противном случае.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **hello**<br>ell | **YES** |
