# Задача 3.2. Минимум и максимум массива

* **Тема:** Массивы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел. Найдите минимальный и максимальный элементы.

## Формат входных данных
В первой строке записано число $N$ ($1 \le N \le 10^5$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$).

## Формат выходных данных
Выведите два числа через пробел: минимальный и максимальный элементы массива.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **5**<br>3 1 4 1 5 | **1 5** |
