# Задача 3.5. Перевёрнутый массив

* **Тема:** Массивы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел. Выведите его элементы в обратном порядке.

## Формат входных данных
В первой строке записано число $N$ ($1 \le N \le 10^5$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^9 \le a_i \le 10^9$).

## Формат выходных данных
Выведите $N$ чисел через пробел — элементы массива в обратном порядке.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **5**<br>1 2 3 4 5 | **5 4 3 2 1** |
