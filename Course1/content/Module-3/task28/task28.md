# Задача 3.8. Палиндром

* **Тема:** Строки
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дана строка $S$, состоящая из строчных латинских букв. Определите, является ли она палиндромом (читается одинаково слева направо и справа налево).

## Формат входных данных
В единственной строке записана строка $S$ ($1 \le |S| \le 10^5$).

## Формат выходных данных
Выведите `YES`, если строка является палиндромом, и `NO` в противном случае.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| abcba | YES |
| hello | NO |
