# Задача 3.1. Сумма элементов массива

* **Тема:** Массивы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел. Вычислите сумму всех его элементов.

## Формат входных данных
В первой строке записано число $N$ ($1 \le N \le 10^5$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^4 \le a_i \le 10^4$).

## Формат выходных данных
Выведите одно число — сумму элементов массива.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **5**<br>1 2 3 4 5 | **15** |
