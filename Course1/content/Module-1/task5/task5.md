# Задача 1.5. Среднее арифметическое

* **Тема:** Арифметика, вещественные числа
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дан массив из $N$ целых чисел. Вычислите их среднее арифметическое.

## Формат входных данных
В первой строке записано число $N$ ($1 \le N \le 1000$).
Во второй строке записаны $N$ целых чисел $a_i$ ($-10^6 \le a_i \le 10^6$).

## Формат выходных данных
Выведите среднее арифметическое с точностью 6 знаков после запятой.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| **3**<br>10 20 30 | **20.000000** |
