# Задача 1.2. Произведение трёх чисел

* **Тема:** Ввод-вывод, арифметика
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны три целых числа $a$, $b$ и $c$. Выведите их произведение.

## Формат входных данных
В единственной строке записаны три целых числа $a$, $b$ и $c$ ($-10^4 \le a, b, c \le 10^4$), разделённые пробелами.

## Формат выходных данных
Выведите одно целое число — произведение $a \cdot b \cdot c$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 2 3 4 | 24 |
