# Задача 1.10. Формула Герона

* **Тема:** Арифметика, формулы
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны длины трёх сторон треугольника $a$, $b$ и $c$. Вычислите его площадь по формуле Герона.

$$S = \sqrt{p(p-a)(p-b)(p-c)}$$

где $p = \frac{a + b + c}{2}$ — полупериметр.

## Формат входных данных
В единственной строке записаны три натуральных числа $a$, $b$, $c$ ($1 \le a, b, c \le 1000$). Гарантируется, что треугольник с такими сторонами существует.

## Формат выходных данных
Выведите вещественное число — площадь треугольника с точностью 6 знаков после запятой.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 3 4 5 | 6.000000 |
