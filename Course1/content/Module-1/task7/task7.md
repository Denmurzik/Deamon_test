# Задача 1.7. Последняя цифра числа

* **Тема:** Арифметика, остаток от деления
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано натуральное число $N$. Выведите его последнюю цифру.

## Формат входных данных
В единственной строке записано натуральное число $N$ ($1 \le N \le 10^9$).

## Формат выходных данных
Выведите одну цифру — последнюю цифру числа $N$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 12345 | 5 |
