# Задача 2.8. Подсчёт цифр числа

* **Тема:** Циклы, деление
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано неотрицательное целое число $N$. Подсчитайте количество цифр в его десятичной записи.

## Формат входных данных
В единственной строке записано число $N$ ($0 \le N \le 10^9$).

## Формат выходных данных
Выведите количество цифр числа $N$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 12345 | 5 |

## Примечания
Число 0 состоит из одной цифры.
