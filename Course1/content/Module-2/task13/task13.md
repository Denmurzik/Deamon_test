# Задача 2.3. Високосный год

* **Тема:** Условный оператор, логические выражения
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано натуральное число $Y$ — номер года. Определите, является ли год високосным.

Год является високосным, если он делится на 4, но не делится на 100, либо делится на 400.

## Формат входных данных
В единственной строке записано натуральное число $Y$ ($1 \le Y \le 10^4$).

## Формат выходных данных
Выведите `YES`, если год високосный, и `NO` в противном случае.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 2000 | YES |
| 1900 | NO |
