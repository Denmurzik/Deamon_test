# Задача 2.9. Реверс числа

* **Тема:** Циклы, деление с остатком
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано натуральное число $N$. Выведите число, полученное записью цифр числа $N$ в обратном порядке. Ведущие нули в результате не выводятся.

## Формат входных данных
В единственной строке записано натуральное число $N$ ($1 \le N \le 10^9$).

## Формат выходных данных
Выведите число $N$, записанное в обратном порядке.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 12345 | 54321 |
| 100 | 1 |
