# Задача 2.1. Максимум из двух чисел

* **Тема:** Условный оператор
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Даны два целых числа $a$ и $b$. Выведите наибольшее из них.

## Формат входных данных
В единственной строке записаны два целых числа $a$ и $b$ ($-10^9 \le a, b \le 10^9$).

## Формат выходных данных
Выведите одно число — максимум из $a$ и $b$.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 3 7 | 7 |
