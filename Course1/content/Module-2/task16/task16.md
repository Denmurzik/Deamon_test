# Задача 2.6. Числа Фибоначчи

* **Тема:** Циклы, рекуррентные соотношения
* **Входной файл:** input.txt
* **Выходной файл:** output.txt
* **Ограничение по времени:** 1.0 секунда
* **Ограничение по памяти:** 256 МБ

## Условие задачи
Дано натуральное число $N$. Вычислите $N$-е число Фибоначчи.

Последовательность Фибоначчи определяется так:
$$F_1 = 1, \quad F_2 = 1, \quad F_n = F_{n-1} + F_{n-2} \text{ при } n \ge 3$$

## Формат входных данных
В единственной строке записано натуральное число $N$ ($1 \le N \le 45$).

## Формат выходных данных
Выведите $N$-е число Фибоначчи.

## Пример

| input.txt | output.txt |
| :--- | :--- |
| 7 | 13 |
