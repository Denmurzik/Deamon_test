# Динамическое программирование

## Основные идеи

**Динамическое программирование (DP)** — метод решения задач путём разбиения на подзадачи с запоминанием результатов.

### Два подхода

1. **Сверху вниз (мемоизация)** — рекурсия + кэширование
2. **Снизу вверх (табуляция)** — итеративное заполнение таблицы

## Пример: Числа Фибоначчи

### Наивная рекурсия — O(2^n)
```python
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)
```

### Мемоизация — O(n)
```python
from functools import lru_cache

@lru_cache(maxsize=None)
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)
```

### Табуляция — O(n), O(1) памяти
```python
def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

## Классические задачи DP

- **Задача о рюкзаке** (Knapsack)
- **Наибольшая общая подпоследовательность** (LCS)
- **Наибольшая возрастающая подпоследовательность** (LIS)
- **Задача о размене монет** (Coin Change)
