# Задача 4А. Размен монет

Дан набор номиналов монет и сумма S. Найдите минимальное количество монет, необходимое для размена суммы S.

## Входные данные
- Первая строка: N — количество номиналов, S — целевая сумма
- Вторая строка: N чисел — номиналы монет

## Выходные данные
Одно число — минимальное количество монет. Если размен невозможен, выведите -1.

## Пример
**Вход:**
```
3 11
1 5 6
```
**Выход:**
```
3
```
*Объяснение: 11 = 5 + 5 + 1*
