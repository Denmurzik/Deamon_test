# Задача 4B. Наибольшая возрастающая подпоследовательность (LIS)

Дана последовательность из N чисел. Найдите длину наибольшей строго возрастающей подпоследовательности.

## Входные данные
- Первая строка: N (1 ≤ N ≤ 10^5)
- Вторая строка: N целых чисел

## Выходные данные
Одно число — длина LIS.

## Пример
**Вход:**
```
6
3 1 4 1 5 9
```
**Выход:**
```
4
```
*Объяснение: подпоследовательность [1, 4, 5, 9]*
