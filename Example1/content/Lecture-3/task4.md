# Задача 3А. Компоненты связности

Дан неориентированный граф с N вершинами и M рёбрами. Найдите количество компонент связности.

## Входные данные
- Первая строка: два числа N и M (1 ≤ N ≤ 10^5, 0 ≤ M ≤ 10^5)
- Следующие M строк: два числа u, v — ребро графа

## Выходные данные
Одно число — количество компонент связности.

## Пример
**Вход:**
```
5 3
1 2
3 4
4 5
```
**Выход:**
```
2
```
